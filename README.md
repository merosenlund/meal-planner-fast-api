# meal-planner-fast-api

Endpoints:
- recipes
  - list of recipes
  - a single recipe
  - create a recipe
  - mark a recipe as in active
  - update a recipe
- meals
  - list of meals
  - single meal
  - create a meal
  - get single meal pdf
  - get list of meals pdf
- orders
  - list of orders
  - single order
  - add meals to order
  - mark order as complete
  - get order pdf
  - get orders pdf
- ingredients
  - create an ingredient
  - delete an ingredient
  - update an ingredient
